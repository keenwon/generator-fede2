使用yo快速构建[fede2](https://github.com/keenwon/fede2)

## Usage
安装yo、generator-fede2

    npm install -g yo generator-fede2

构建项目

	mkdir your-project-dir
	cd your-project-dir
	yo fede2

运行

	# 测试
	npm test

	# 编译：测试环境
	npm run build:testing
	
	# 编译：生产环境
	npm run build:production
